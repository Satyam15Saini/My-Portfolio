# 👨‍💻 Satyam Saini - Software Development Engineer Portfolio

Welcome to the repository for my personal portfolio website! This project is designed to showcase my skills, projects, and experience as a Software Development Engineer. It features a modern, premium UI with a custom glassmorphic design and interactive animations.

> **🚀 Live Preview:** [Check out the live portfolio here!](https://satyam15saini.github.io/My-Portfolio/)

---

## ✨ Key Features

* **Modern UI/UX:** Premium dark-theme design with a vibrant teal glassmorphic navigation bar and overlays.
* **Interactive Animations:** Features a custom preloader, floating 3D elements, dynamic typing animations, and scroll-triggered reveals using Intersection Observer.
* **Dynamic Background:** A custom interactive particle background built with HTML5 Canvas.
* **Project Filtering:** Built-in JavaScript filtering system to sort projects by category (AI/ML, Full Stack, DevOps).
* **Fully Responsive:** Optimized for a seamless experience across desktop, tablet, and mobile devices.
* **Easter Egg:** Try typing the Konami Code (`↑` `↑` `↓` `↓` `←` `→` `←` `→` `B` `A`) on the live site! 🎮

---

## 🛠️ Tech Stack

This portfolio is built using pure, vanilla web technologies to ensure maximum performance and zero dependencies:

* **HTML5:** Semantic structuring and accessible markup.
* **CSS3:** Custom variables, CSS Grid/Flexbox, complex keyframe animations, glassmorphism (`backdrop-filter`), and responsive media queries.
* **Vanilla JavaScript:** DOM manipulation, Intersection Observers for lazy/scroll loading, particle generation, and event handling.
* **Icons:** Official brand SVGs from [Devicon](https://devicon.dev/).
